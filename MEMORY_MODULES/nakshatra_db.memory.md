# Module: nakshatra_db.py
## Last Updated: 2026-03-02

## PURPOSE
Database of nakshatra metadata beyond the basic config.py entries. Contains mythology, deity associations, nature (Deva/Manushya/Rakshasa), Gana (Deva/Manushya/Rakshasa), Varna, Yoni, Nadi, and Tarabala sequence data for each of the 27 (+ Abhijit) nakshatras.

## KEY FUNCTIONS
No functions — pure data lookup tables.

## IMPORTANT CONSTANTS
For each of 27 nakshatras: name, lord, deity, nature, gana, varna, yoni_animal, nadi, tarabala_sequence

## DEPENDENCIES
config.py
